# 文枢 wenshu — 页面生成执行文档

> 本文档是一份 AI 可直接执行的指令集。AI 读取本文档后，能够生成「文枢」智能写作平台的完整前端页面。
> 输出为单文件 HTML（内含 CSS + JS），可直接在浏览器打开运行。

---

## 执行总指令

```
请根据以下规范，生成一个名为 wenshu-app.html 的单文件 Web 应用。
技术栈：纯 HTML5 + CSS3 + Vanilla JavaScript，不依赖任何外部库。
整体架构：单页应用（SPA），通过 JS 控制页面切换，共包含以下页面/视图：

## 一、全局设计规范

### 1.1 主题系统

```
实现双主题切换（暗色默认 + 亮色）：

CSS 变量定义在 :root（暗色主题）：
--bg-primary: #0a0a0f（主背景，近纯黑带微蓝）
--bg-secondary: #12121a（卡片/面板背景）
--bg-tertiary: #1a1a2e（输入框/悬浮背景）
--text-primary: #e8e8f0（主文字，微冷白）
--text-secondary: #8888a0（次要文字）
--text-muted: #555570（占位符/禁用文字）
--accent: #6366f1（主强调色，靛蓝）
--accent-hover: #818cf8（强调色悬浮）
--border: #2a2a3e（边框线）
--success: #10b981
--warning: #f59e0b
--error: #ef4444

亮色主题通过 [data-theme="light"] 覆盖：
--bg-primary: #f8f9fc
--bg-secondary: #ffffff
--bg-tertiary: #f0f1f5
--text-primary: #1a1a2e
--text-secondary: #5a5a7a
--text-muted: #9a9ab0
--accent: #4f46e5
--accent-hover: #6366f1
--border: #e2e4ea

主题切换按钮放在每个页面的右上角，点击切换 document.documentElement 的 data-theme 属性。
```

### 1.2 全局样式规范

```
字体：font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif
圆角：小元素 6px，卡片 12px，弹窗 16px
过渡：所有交互元素添加 transition: all 0.2s ease
按钮风格：扁平化，无 box-shadow 发光效果，无 translateY 悬浮动画
  - 主按钮：背景 var(--accent)，白色文字，hover 时背景变 var(--accent-hover)
  - 次按钮：背景 transparent，border 1px solid var(--border)，hover 时背景 var(--bg-tertiary)
  - 文字按钮：无背景无边框，hover 时 opacity 0.7
输入框：背景 var(--bg-tertiary)，border 1px solid var(--border)，focus 时 border-color 变 var(--accent)
滚动条：自定义细滚动条（6px 宽，圆角，半透明轨道）
```

### 1.3 页面切换机制

```javascript
// 所有页面容器用 class="page" 包裹，默认 display:none
// 当前页面添加 class="active"（display:block/flex）
// 切换函数：
function showPage(pageId) {
  document.querySelectorAll('.page').forEach(p => p.classList.remove('active'));
  document.getElementById(pageId).classList.add('active');
}
```
